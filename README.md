## General info
This project is simple Lorem ipsum dolor generator.
	
## Technologies Used:
* Front-end: React-redux
* Back-end: NodeJS, express
* Database: MongoDB
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```
